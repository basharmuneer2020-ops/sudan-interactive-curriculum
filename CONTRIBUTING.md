## أنواع المساهمات

### 🔬 1. إنشاء معمل تفاعلي جديد (الأكثر تأثيراً)
المعامل هي قلب المشروع — محاكاة تفاعلية تحل محل المعمل الحقيقي.

**المطلوب لكل معمل:**
- مكون React تفاعلي (`.jsx`)
- صفحة توجيه (`page.js`)
- ربط المعمل في صفحة الصف الدراسي
- اختبار على الجوال والشاشات الصغيرة

**الأولوية:** معامل الأول والثاني ثانوي ← ثم المتوسطة ← ثم الابتدائية

### 📝 2. إضافة محتوى تعليمي
- شروحات الدروس والفصول
- أسئلة تفاعلية ومراجعات
- ملخصات الوحدات

### 🎨 3. تحسين التصميم والواجهة
- تحسين تجربة المستخدم (UX)
- دعم أفضل للجوال
- تحسين الأداء على الأجهزة الضعيفة

### 🐛 4. إصلاح الأخطاء
- أخطاء في المحتوى العلمي
- أخطاء تقنية في الكود
- مشاكل التوافق مع المتصفحات

### 📖 5. التوثيق والترجمة
- تحسين التوثيق
- ترجمة الواجهة للإنجليزية (اختياري)

## كيف تختار مهمة؟

### للمبتدئين 🟢
ابحث عن Issues معلّمة بـ `good first issue` — مهام بسيطة مثل:
- إصلاح أخطاء إملائية
- تحسين تنسيق صفحة موجودة
- إضافة محتوى نصي لدرس

### للمتوسطين 🟡
ابحث عن Issues معلّمة بـ `help wanted` — مهام مثل:
- إنشاء صفحة صف جديد بمحتوى كامل
- تحسين مكون تفاعلي موجود
- إضافة ميزة جديدة

### للمتقدمين 🔴
- إنشاء معمل تفاعلي كامل
- بناء نظام التقييم والألعاب التعليمية
- تحسين الأداء والـ SEO

### خريطة المهام حسب المادة

| المادة | المعامل الجاهزة | المعامل المطلوبة | مستوى الصعوبة |
|--------|----------------|-----------------|--------------|
| 🧪 الكيمياء | التحليل الكيفي (ث3) | عضوية (ث2)، تفاعلات (ث1) | متوسط-متقدم |
| ⚛️ الفيزياء | الجاذبية (ث3) | كهرومغناطيسية (ث2)، بصريات (ث1) | متقدم |
| 📐 الرياضيات | التفاضل (ث3) | تكامل (ث3)، هندسة (ث2) | متوسط |
| 🧬 الأحياء | وراثة مندل (ث3) | خلية (ث2)، تشريح (ث1) | متوسط-متقدم |

## سير العمل

### الخطوات الأساسية

</div>

```bash
# 1. انسخ المشروع (Fork) على حسابك في GitHub

# 2. استنسخه محلياً
git clone https://github.com/YOUR-USERNAME/sudan-interactive-curriculum.git

# 3. أنشئ فرع جديد لمهمتك
git checkout -b feature/اسم-المهمة
# أمثلة:
# git checkout -b feature/chemistry-organic-lab
# git checkout -b fix/mobile-responsive
# git checkout -b content/grade2-physics-chapters

# 4. اعمل تعديلاتك واحفظها
git add .
git commit -m "وصف واضح للتعديل"

# 5. ارفع الفرع
git push origin feature/اسم-المهمة

# 6. افتح Pull Request على GitHub
```

<div dir="rtl" align="right">

### تسمية الفروع

| النوع | الصيغة | مثال |
|-------|--------|------|
| ميزة جديدة | `feature/وصف` | `feature/chemistry-organic-lab` |
| إصلاح خطأ | `fix/وصف` | `fix/mobile-menu-rtl` |
| محتوى | `content/وصف` | `content/grade2-biology-chapters` |
| تحسين | `improve/وصف` | `improve/homepage-performance` |

### رسائل الـ Commit

</div>

```
# الصيغة:
نوع: وصف مختصر

# أمثلة:
feat: إضافة معمل الكيمياء العضوية للثاني ثانوي
fix: إصلاح عرض المعادلات على شاشة الجوال
content: إضافة فصول الفيزياء للأول ثانوي
style: تحسين تصميم بطاقات المراحل
docs: تحديث دليل المساهمة
```

## معايير الكود

### التقنيات المستخدمة
- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS
- **Language:** JavaScript/JSX
- **3D/Simulation:** Three.js (للمعامل ثلاثية الأبعاد)

### قواعد أساسية

1. **العربية أولاً** — كل النصوص المرئية بالعربية، اتجاه RTL
2. **الجوال أولاً** — صمم للشاشات الصغيرة ثم وسّع
3. **الأداء** — تجنب المكتبات الثقيلة، حسّن الصور
4. **البساطة** — كود واضح وسهل الفهم للمساهمين الجدد
5. **إمكانية الوصول** — ألوان واضحة، خطوط كبيرة كفاية

### نموذج مكون معمل تفاعلي

</div>

```jsx
'use client'
import { useState } from 'react'

export default function LabName() {
  // 1. حالة المكون

  // 3. واجهة المعمل

    <div className="min-h-screen bg-gray-50" dir="rtl">

      </main>
    </div>
  )
}
```

## إرشادات المحتوى التعليمي

### المصدر الرسمي
كل المحتوى يجب أن يتبع منهج **المركز القومي للمناهج والبحث التربوي** — وزارة التربية والتعليم السودانية.

### معايير المعمل التفاعلي الجيد
1. **يحل مشكلة حقيقية** — شيء الطالب ما بيشوفه في المدرسة
2. **تفاعلي** — الطالب يتحكم ويجرب بنفسه
3. **متدرج** — من السهل للصعب
4. **مرئي** — ألوان وحركة توضح المفهوم
5. **سوداني السياق** — أمثلة من الحياة اليومية السودانية

## نظام الإصدارات

نتبع نظام **Semantic Versioning** (`vMAJOR.MINOR.PATCH`):

| الرقم | متى يتغير | مثال |
|-------|----------|------|
| MAJOR | تغيير جذري في الهيكل | v1.0.0 → v2.0.0 |
| MINOR | إضافة معمل أو مرحلة جديدة | v1.0.0 → v1.1.0 |
| PATCH | إصلاح خطأ أو تحسين بسيط | v1.0.0 → v1.0.1 |

### الإصدار الحالي: `v1.0.0`
- ✅ فهرسة كل المراحل (روضة — ثانوي)
- ✅ 4 معامل تفاعلية (الثالث ثانوي)
- ✅ الصفحة الرئيسية الشاملة
