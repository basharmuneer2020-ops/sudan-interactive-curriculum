# المساهمة في المنهج السوداني التفاعلي

شكراً لاهتمامك بالمساهمة! هذا المشروع صدقة جارية ونرحب بأي مساهمة.

## كيف تساهم؟

### إضافة معمل جديد

1. أنشئ مكون React في المجلد المناسب: `src/components/{المادة}/`
2. أنشئ صفحة التوجيه في: `src/app/{المرحلة}/{الصف}/{المادة}/{الوحدة}/page.js`
3. أضف المعمل في قائمة المعامل في صفحة الصف

### تسمية الملفات

</div>

```
src/components/{subject}/ComponentName.jsx    # PascalCase for components
src/app/{stage}/{grade}/{subject}/{unit}/page.js  # kebab-case for routes
```

<div dir="rtl" align="right">

### المراحل والصفوف

| المسار | المرحلة |
|--------|---------|
| `secondary/grade-3` | الصف الثالث الثانوي |
| `secondary/grade-2` | الصف الثاني الثانوي |
| `secondary/grade-1` | الصف الأول الثانوي |
| `middle/grade-3` | الصف الثالث المتوسط |
| `primary/grade-*` | المرحلة الأساسية |

### المواد

| المسار | المادة |
|--------|--------|
| `biology` | الأحياء |
| `chemistry` | الكيمياء |
| `physics` | الفيزياء |
| `math` | الرياضيات |

## معايير الجودة

- الواجهة بالعربية مع دعم RTL
- يجب أن تعمل على الأجهزة الضعيفة والشاشات الصغيرة
- المحتوى يتبع المنهج الرسمي لوزارة التربية والتعليم السودانية
