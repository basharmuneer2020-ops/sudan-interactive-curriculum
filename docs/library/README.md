# 📚 مكتبة المقررات الدراسية — المنهج السوداني

فهرس شامل لكتب المنهج السوداني بصيغة PDF لكل المراحل الدراسية.

## تسمية الملفات

استخدم نظام الترميز:

</div>

```
{STAGE}-{GRADE}-{SUBJECT}.pdf

أمثلة:
SEC-G12-CHE.pdf    ← كيمياء الثالث ثانوي
SEC-G12-PHY.pdf    ← فيزياء الثالث ثانوي
SEC-G11-BIO.pdf    ← أحياء الثاني ثانوي
MID-G9-MAT.pdf     ← رياضيات التاسع
PRI-G6-SCI.pdf     ← علوم السادس ابتدائي
```

<div dir="rtl" align="right">

## كيف تستخدم المكتبة؟

1. **حمّل المجلدات** من روابط Google Drive أعلاه
2. **أعد تسمية الملفات** حسب نظام الترميز
3. **احفظها** في المجلد المناسب داخل `docs/library/`
4. **حدّث الفهرس** في `CATALOG.md`

> ⚠️ ملفات PDF مضافة في `.gitignore` — لا تُرفع على GitHub
