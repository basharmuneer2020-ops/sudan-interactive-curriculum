# 🤖 دليل استخدام الذكاء الاصطناعي في المساهمة

## لماذا هذا الدليل؟

هذا المشروع تم بناؤه بالكامل بمساعدة **Claude** (من Anthropic). نشجع كل المساهمين على استخدام أدوات الذكاء الاصطناعي لتسريع عملهم وتحسين جودته.

---

## الأدوات المتاحة

### 1. Claude Desktop (Cowork Mode)
**الأفضل لـ:** بناء المعامل التفاعلية والصفحات الكاملة

كيف تستخدمه:
- حمّل [Claude Desktop](https://claude.ai/download)
- افتح وضع Cowork
- اربط مجلد المشروع
- اطلب من Claude بناء المعمل أو الصفحة

**مثال للطلب:**

</div>

```
أنا أعمل على مشروع المنهج السوداني التفاعلي.
محتاج أبني معمل تفاعلي للكيمياء العضوية — الصف الثاني ثانوي، الباب الأول.
الكود: SEC-G11-CHE-C01-LAB-01

المعمل يحاكي تفاعلات المركبات العضوية.
المنهج يغطي: الألكانات، الألكينات، الألكاينات، والمجموعات الوظيفية.

استخدم React + Tailwind CSS. الواجهة بالعربية RTL.
الملف يُحفظ في: src/components/chemistry/OrganicChemLab.jsx
صفحة التوجيه في: src/app/secondary/grade-2/chemistry/organic/page.js
```

<div dir="rtl" align="right">

### 2. Claude Code (Terminal)
**الأفضل لـ:** المطورين المتقدمين الذين يفضلون سطر الأوامر

</div>

```bash
# تثبيت Claude Code
npm install -g @anthropic-ai/claude-code

cd sudan-interactive-curriculum
claude

# ثم اطلب ما تحتاجه
> أنشئ معمل فيزياء للكهرومغناطيسية SEC-G11-PHY-C02-LAB-01
```

<div dir="rtl" align="right">

### 3. Claude.ai (الموقع)
**الأفضل لـ:** كتابة المحتوى التعليمي والشروحات

يمكنك استخدام Claude عبر المتصفح لـ:
- كتابة شروحات الدروس
- إنشاء أسئلة تفاعلية
- مراجعة المحتوى العلمي
- ترجمة أو تحسين النصوص

## نصائح للحصول على أفضل النتائج

### 1. أعطِ سياقاً كافياً
- اذكر اسم المشروع ونظام الترميز
- حدد المادة والصف والباب
- اشرح ماذا يحاكي المعمل
- حدد التقنيات (React, Tailwind, Three.js)

### 2. استخدم نظام الترميز

</div>

```
المعنى: الثانوية → الثاني ثانوي → فيزياء → الباب 2 → معمل → رقم 01
```
